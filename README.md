# 🎓 LinkedIF

Aplicação web desenvolvida para reunir informações sobre o curso de Tecnologia em Sistemas para Internet do IFSP e apresentar um perfil acadêmico e profissional do estudante.

O projeto foi desenvolvido de forma autoral com o objetivo de praticar desenvolvimento web, organização de interfaces e construção de páginas responsivas.

---

## 📖 Sobre o projeto

O LinkedIF foi criado para centralizar informações relevantes do curso de Tecnologia em Sistemas para Internet do IFSP, como objetivo do curso, matriz curricular e informações institucionais.

Além disso, a aplicação possui uma área dedicada ao perfil do estudante, reunindo dados de contato, competências, formação acadêmica, experiência profissional e objetivo de carreira em um único ambiente.

---

## ✨ Funcionalidades

- ✅ Página inicial do estudante
- ✅ Perfil profissional
- ✅ Informações do curso
- ✅ Matriz curricular
- ✅ Competências técnicas e pessoais
- ✅ Formação acadêmica
- ✅ Experiência profissional
- ✅ Página de contato
- ✅ Layout responsivo
- ✅ Navegação entre páginas
